About The Detector

        
This tool uses a machine learning model to classify emails as spam or ham.

        
This Model is trained On large Dataset  (by Enron) 

        
We have used TF-IDF for classification.

        
Its Probabilistic Model which calculates the probability of a message being spam or ham.

        
This Model is very accurate because of its diverse Dataset
